ViewCVS: Viewing CVS Repositories

The ViewCVS software was inspired by cvsweb (by Henner Zeller). I wanted to make some changes and updates, but cvsweb was implemented in Perl (and rather poorly, IMO). So I undertook the task to convert the software to Python.

ViewCVS is currently at version 0.1. It is almost a straight port of the cvsweb script. I made some cleanups and other modifications, based on some of Python's strengths, but there is still some ugly stuff in there. Future revisions will clear that out, along with my work to start adding features. The functionality of ViewCVS is equal to that of cvsweb, minus the "annotation" support. Annotation requires read/write access to the CVS repository (at this time), and I believe that is a Bad Thing to do. One of my tasks will be eliminating the read/write requirement.
